## 💬 ROTEIRO DE ENTREVISTAS

### PERGUNTA 1: Por que está migrando da fisioterapia para dados?  
**Resposta sugerida:**  
Sempre trabalhei com análise de performance e tomada de decisão baseada em dados clínicos. Quero escalar esse impacto utilizando IA.

---

### PERGUNTA 2: O que é overfitting?  
**Resposta sugerida:**  
É quando o modelo aprende muito bem os dados de treino, mas não generaliza para novos dados. Podemos reduzir isso com validação cruzada e regularização.

---

### PERGUNTA 3: Diferença entre regressão e classificação?  
**Resposta sugerida:**  
Regressão prevê valores numéricos contínuos. Classificação prevê categorias.

---

### PERGUNTA 4: Como tratar dados faltantes?  
**Resposta sugerida:**  
Podemos remover, imputar média/mediana ou usar técnicas mais avançadas dependendo do contexto.

---

### PERGUNTA 5: Como proteger dados sensíveis?  
**Resposta sugerida:**  
Utilizando criptografia, controle de acesso e anonimização.
